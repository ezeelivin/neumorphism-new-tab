### Manual Installation
1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension directory
5. Open a new tab to see the extension in action

## Usage

- **Shortcuts**: Click on any shortcut to visit the site, or use the edit button to customize
- **Bookmarks**: Access your Chrome bookmarks directly from the new tab page
- **Settings**: Customize the appearance and behavior through the settings panel
