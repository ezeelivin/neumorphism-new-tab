# Changelog

All notable changes to the Neumorphism NewTab extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2023-12-01

### Added
- Initial release with neumorphism design principles
- Customizable shortcuts to favorite websites
- Clock widget with digital display
- Bookmarks integration with Chrome bookmarks API
- Light/dark mode toggle with system preference detection
- Settings panel for personalization options
- Local storage for saving user preferences
- Responsive design for various screen sizes
- Modern, clean user interface
